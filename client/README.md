# 🎟️ Festix | World-Class Event & Ticketing Ecosystem

Festix is a high-fidelity, full-stack event management platform designed for the 2026 digital economy. Built with a focus on **security**, **scalability**, and **motion-driven UX**, it provides a seamless experience for Attendees, Organizers, and Administrators.

---

## 🚀 Key Features

- **🛡️ Multi-Role Authentication:** Granular access control for Users, Organizers, and Admins.
- **📊 Dynamic Dashboards:** Real-time analytics with Recharts for revenue and user growth.
- **💳 Secure Checkout:** Stripe integration for localized and international payments.
- **📱 Digital Ticket Stubs:** Instant QR code generation with high-contrast scanning optimization.
- **🌓 Adaptive UI:** System-aware Dark Mode with persistent theme context.
- **⚡ High Performance:** Lazy loading, code splitting, and memoized state management.

---

## 🛠️ Tech Stack

### **Frontend Core**

- **React 18:** Functional components with Hooks.
- **Tailwind CSS:** Utility-first styling for high-speed UI development.
- **Material UI (MUI):** Premium component library for complex dashboard elements.
- **Framer Motion:** High-end page transitions and micro-interactions.
- **Lucide React:** Consistent, lightweight iconography.

### **State & Routing**

- **React Router 6:** Nested routing and declarative protection.
- **Context API:** Decentralized state for Auth, Theme, and Data.
- **Axios:** Interceptor-based API communication.
